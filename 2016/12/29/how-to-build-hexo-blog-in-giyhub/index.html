利用hexo在Github上搭建自己的博客

发表于 2016-12-29 | 分类于 技术 |

前言

      作为一个程序猿，其实早就想拥有一个属于自己的个人网站，奈何总会有这样那样的事情（拖延症）导致计划一直延后。前些天终于有时间把自己的个人博客搭建了起来，使用的是hexo博客框架。为什么选择hexo呢，主要是因为hexo的文档清晰，易读，基本上看一下官方文档就基本可以懂得怎么搭建，再加上使用的是node.js这个工具个人比较熟悉。其实相关的教程网上很多，看一下也很简单，写这篇文章的主要是为了防止自己以后忘了整个流程。

准备工作

安装Node

由于hexo 是基于node.js 环境来进行生成搭建博客页面的，所以需要我们的电脑安装node.js 话不多说传送门进入node.js官网。node.js官网一般推荐两个版本，分别为通用版和最新版（我推荐通用版吧），点击就可以下载。下载好后，按照正常安装软件的方法安装即可。mac用户跟windows用户都差不多一样。
安装好了以后在命令行工具输入：

安装Git

git 主要用于本地生成好的博客文件上传至github仓库，如果是mac用户的话就无需安装了，因为系统自带有git，以下主要针对windows用户，点击这里进入git的下载页面，选择对应的系统下载即可，下载后还是按照老套路来进行安装，如果搞不懂git的话请自行百度，网上有很多教程，在这里就不多说了。
